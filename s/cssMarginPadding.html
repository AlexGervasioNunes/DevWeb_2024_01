İntrodução


Nesta página, iremos aprender um pouco sobre o HTML e o CSS. Os dois oferecem um alto grau de flexibilidade e personalização na criação de páginas da web. Com essas linguagens, é possível criar layouts responsivos e adaptáveis para dispositivos móveis, personalizar a aparência dos elementos de uma página, definir cores, fontes e estilos, entre outras possibilidades.




HTML


O que é "HTML"?

O HTML (Linguagem de Marcação de HiperTexto) é o bloco de construção mais básico da web. Define o significado e a estrutura do conteúdo da web. Outras tecnologias além do HTML geralmente são usadas para descrever a aparência/apresentação (CSS) ou a funcionalidade/comportamento (JavaScript) de uma página da web.
A palavra "Hipertexto" refere-se aos links que conectam páginas da Web entre si, seja dentro de um único site ou entre sites. Links são um aspecto fundamental da web. Ao carregar conteúdo na Internet e vinculá-lo a páginas criadas por outras pessoas, você se torna um participante ativo na "World Wide Web".

CSS


O que é "CSS"?


CSS é a sigla para o termo em inglês Cascading Style Sheets que, traduzido para o português, significa Folha de Estilo em Cascatas. O CSS é fácil de aprender e entender e é facilmente utilizado com as linguagens de marcação HTML ou XHTML. O CSS é usado para estilizar elementos escritos em uma linguagem de marcação como HTML. O CSS separa o conteúdo da representação visual do site. Pense na decoração da sua página.
