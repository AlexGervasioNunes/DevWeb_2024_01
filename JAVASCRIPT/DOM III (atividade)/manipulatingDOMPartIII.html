Métodos JavaScript




Método querySelector()


O JavaScript QuerySelector é uma ferramenta extremamente útil para selecionar elementos no DOM com simplicidade e eficiência. Ele permite realizar uma infinidade de operações de manipulação e interação com os elementos de uma página web. Com o QuerySelector, podemos modificar estilos, conteúdos, adicionar ou remover classes, realizar eventos e muito mais.


Além disso, o JavaScript QuerySelector possui recursos avançados, como seleção com base em seletor CSS, elementos pai, filhos e irmãos, e atributos dos elementos. Esses recursos adicionam flexibilidade e poder ao desenvolvimento web com JavaScript.


Em resumo, o JavaScript QuerySelector é uma ferramenta indispensável para qualquer desenvolvedor web que deseja criar interações dinâmicas e personalizadas em suas páginas. Selecione elementos no DOM com facilidade e transforme suas aplicações web em experiências interativas e envolventes para os usuários.




Método querySelectorAll()


O método querySelectorAll é uma função incorporada em JavaScript que retorna uma NodeList estática (não uma matriz real) de elementos no documento que correspondem a um grupo de seletores de CSS específicos. A lista retornada incluirá todos os elementos descendentes que correspondem aos seletores fornecidos.


O querySelectorAll() recebe uma string em formato de seletor CSS e retorna uma nodelist com esses elementos. A linha document.querySelectorAll('.product-image') busca por todos os elementos com a classe product-image. Logo, quanto maior a DOM, maior o tempo de execução de cada uso desse método.


A função querySelectorAll() recebe um argumento de string contendo um seletor CSS e retorna um objeto NodeList, representando os elementos do documento que correspondem ao seletor. Se nenhum elemento coincide, a função retorna um objeto NodeList vazio. Se a string do seletor é inválida, querySelectorAll() lançará um exceção.
